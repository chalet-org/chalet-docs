Build configurations in Chalet control some of the key properties of the build, like the optimization level, debug symbols, sanitizers, and profiling - anything that requires a distinct build.

\\

## [Default Configurations]

Chalet contains several default build configurations:


```json
"defaultConfigurations": [
   "Release",
   "Debug",
   "RelWithDebInfo",
   "MinSizeRel",
   "RelStable",
   "Profile"
   "DebugSanitize",
   "DebugSanitizeAddress",
   "DebugSanitizeThread",
   "DebugSanitizeMemory",
   "DebugSanitizeLeak",
   "DebugSanitizeUndefined",
   "DebugSanitizeHW",
   "DebugSanitizeHWAddress"
]
```
