Basic support for in-project packages has been added as of version 0.7.0. As further features get added, they will be documented here. In-project packages are a way of isolating relevant build settings, and importing them to source targets as needed.

\\

## [Overview]

Packages are designed in such a way that is meant to be independent of the method used to build a target.
